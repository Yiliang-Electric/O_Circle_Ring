# O_Circle_Ring

Tracking and Counting
This project is used to detect, track, and count small ring-shaped objects moving on a conveyor belt using computer vision.

The tracking method are base on 
1. https://docs.ultralytics.com/modes/track/#python-examples.
2. https://pypi.org/project/deep-sort-realtime

## Features

- Detect small rings on a conveyor
- Track rings in real time
- Count rings automatically
- Reduce manual counting errors
- Display total count on screen
